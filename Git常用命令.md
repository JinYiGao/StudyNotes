# Git常用命令

```shell
git init  #初始化 

git add .  #添加所有 

git commit -m "first commit"   #提交 并添加注释

git remote add origin https://github.com/JinYiGao/Geological-hazard-weapon  #远程连接到仓库

git push -u origin master  #push到仓库
#  若要提交到分支 则 
#  git push -u origin 分支名

git checkout -b 分支名    #（你自己创建的分支名）

```
